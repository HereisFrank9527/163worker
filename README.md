# 163worker - 网易云音乐下载器 & NCM转换器

一个基于PyQt-Fluent-Widgets开发的现代化网易云音乐下载器和NCM格式转换器，采用Fluent Design设计风格，提供优雅的用户界面和流畅的操作体验。

[项目地址](https://github.com/HereisFrank9527/163worker)

## 功能特点

### 🎵 网易云音乐下载
- 支持通过歌单ID下载整个歌单
- 支持多种音质选择
- 可设置下载速度限制
- 实时显示下载进度
- 支持暂停/继续下载

### 📁 NCM格式转换
- 支持批量转换NCM格式文件为MP3格式
- 自动读取NCM文件元数据
- 支持自定义输出目录
- 实时显示转换进度

### 🔧 工具功能
- API可用性自动检测
- 手动API检查功能
- 实时日志记录
- 自动检查最新版本
- 优雅的错误提示

### 🎨 界面设计
- 采用Fluent Design设计风格

## 技术栈

- **Python 3.9** - 开发语言
- **PyQt5** - GUI框架
- **PyQt-Fluent-Widgets** - Fluent Design组件库
- **mutagen** - 音频文件处理
- **pycryptodome** - 加密解密
- **ncmdump** - NCM格式转换

## 使用方法


```bash
pip install -r requirements.txt
python main.py
```

### 歌单下载

1. 在左侧导航栏选择「歌单下载」
2. 输入网易云音乐歌单ID
3. 选择所需音质
4. 设置下载保存路径（默认：`./downloads`）
5. 点击「开始下载」按钮
6. 可随时点击「停止下载」暂停下载

### NCM格式转换

1. 在左侧导航栏选择「NCM转换」
2. 设置源文件夹（包含NCM文件）
3. 设置目标文件夹（转换后MP3文件保存位置，默认：`./trans`）
4. 点击「开始转换」按钮
5. 可随时点击「停止转换」暂停转换

### 查看日志

在左侧导航栏选择「日志」查看详细的操作日志

### 检查API状态

在「歌单下载」页面点击「检查api」按钮，可手动测试API可用性
